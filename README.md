# Detective-OS (Web Prototype)

## Prerequisites
- Node.js 18+ (includes npm)

## Setup
```bash
npm install
```

## Run locally
```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for production
```bash
npm run build
npm run preview
```

## Notes
- Static assets live in `public/` (wallpaper, audio, avatar).
- Main UI logic is in `src/App.jsx`.
